# Advent of Code 2025

* This challenge at [https://adventofcode.com/2025](https://adventofcode.com/2025) consists of a coding challenge for 12 days of December 2025. Each challenge answered correctly gives a gold star.

* The challenges come with a part 2 that gives an additional gold star.

* The goal is to get 24 stars in total by programming in any language of your choosing.
